关于hexo的一些bug.

1. $ hexo deploy ERROR Deployer not found: git

文档说hexo3.0后需要在 _config.yml 中设置 deploy中的type为git（过去是github），

然后，记得一定要在 blog 目录里 执行 npm install hexo-deployer-git –save ，

因为，你一定记得当初…

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
